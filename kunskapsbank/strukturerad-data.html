Strukturerad data – Förbättrar dina chanser att synas på Google

Schema markup, eller strukturerad data, är en teknik som går under Teknisk SEO och hjälper till att förbättra sökmotorers förståelse av webbplatsers innehåll. Genom att lägga till specifika koder och format kan du hjälpa sökmotorer som Google att bättre förstå och presentera ditt innehåll på ett mer relevant och användarvänligt sätt. I den här artikeln kommer vi att gå igenom olika typer av strukturerad data och hur de kan användas för att förbättra din webbplats sökmotoroptimering.

2 exempel på resultat av strukturerad data

Rich snippets

Rich snippets är förbättrade sökresultat som ger mer information om en webbsida, såsom recensioner, priser, evenemang och recept. De gör det enklare för användare att hitta relevant information och ökar chansen att de klickar på ditt sökresultat.

Fördelar med rich snippets
  • Ökad synlighet i sökresultaten
  • Högre klickfrekvens (CTR)
  • Bättre användarupplevelse
Hur skapar man rich snippets?

För att skapa rich snippets måste du använda strukturerad data i ditt innehåll. Det finns flera format för att göra detta, inklusive JSON-LD, Microdata och RDFa, som vi kommer att diskutera mer i detalj nedan.

JSON-LD

JSON-LD (JavaScript Object Notation for Linked Data) är ett lättviktigt dataformat för att strukturera information på webben. Det är enkelt att implementera och läsa av både människor och maskiner.

Användning av JSON-LD för strukturerad data

För att använda JSON-LD i din webbplats kod, måste du lägga till en <script>-tagg av typen "application/ld+json" och inkludera den strukturerade datan i JSON-format.

Fördelar med JSON-LD

  • Enkelt att implementera
  • Lätt att läsa och förstå för både människor och maskiner
  • Separerar strukturerad data från sidans HTML, vilket gör det enklare att underhålla

Microdata

Microdata är en HTML5-specifikation som låter dig lägga till strukturerad data direkt i ditt HTML-innehåll. Det gör det möjligt för sökmotorer att extrahera och förstå strukturerad information från webbsidor.

Implementering av Microdata

För att använda Microdata måste du lägga till itemscope, itemtype och itemprop attribut till dina HTML-taggar. Itemscope definierar att ett element innehåller strukturerad data, medan itemtype anger vilken typ av data det är och itemprop definierar egenskaperna för den datatypen.

Fördelar med Microdata

  • Integrerad direkt i HTML-koden, vilket gör det lätt att koppla strukturerad data med sidans innehåll
  • Kompatibel med HTML5-specifikationen
  • Lätt att läsa för både människor och maskiner

RDFa

RDFa (Resource Description Framework in Attributes) är en teknik för att lägga till strukturerad data i HTML- och XHTML-dokument. Det är en äldre specifikation än Microdata och JSON-LD, men fortfarande användbar för vissa ändamål.

Användning av RDFa för strukturerad data

För att använda RDFa måste du lägga till attribut som vocab, typeof och property till dina HTML-element. Dessa attribut hjälper sökmotorer att förstå och extrahera den strukturerade informationen från din webbsida.

Fördelar med RDFa

  • Fungerar med både HTML- och XHTML-dokument
  • Ger stöd för att använda flera vokabulär samtidigt
  • Kan användas tillsammans med andra strukturerade dataformat som Microdata och JSON-LD

Vanliga frågor om Strukturerad data

Vilket strukturerat dataformat är enklast att implementera och underhålla?

JSON-LD anses vara enklast att implementera och underhålla eftersom det separerar strukturerad data från sidans HTML.

Vilken teknik används för att skapa rich snippets?

Rich snippets skapas genom att använda strukturerad data i ditt innehåll, antingen genom JSON-LD, Microdata eller RDFa.

Vad är skillnaden mellan Microdata och RDF?

Microdata är en HTML5-specifikation som låter dig lägga till strukturerad data direkt i ditt HTML-innehåll, medan RDFa är en äldre teknik som fungerar med både HTML- och XHTML-dokument. De använder olika attribut för att ange strukturerad data och RDFa ger stöd för att använda flera vokabulär samtidigt.

Vilken fördel har Microdata över JSON-LD?

En fördel med Microdata är att det är integrerat direkt i HTML-koden, vilket gör det lätt att koppla strukturerad data med sidans innehåll. JSON-LD å andra sidan separerar strukturerad data från sidans HTML, vilket kan göra det svårare att koppla data till specifika delar av innehållet.

Kan jag använda flera strukturerade dataformat på samma webbsida?

Ja, det är möjligt att använda flera strukturerade dataformat på samma webbsida. Till exempel kan du använda JSON-LD för vissa delar av sidan och Microdata eller RDFa för andra delar. Det är dock viktigt att inte duplicera samma strukturerade data med olika format, eftersom det kan leda till förvirring för sökmotorer.
